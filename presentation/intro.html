Ebony is a web page stylesheets collection, based on ivory, designed originally for ZJU's forum CC98.org(only vaild in ZJU campus).

Ebony is composed of:

  • A responsive grid framework modified from ivory;
  • A CSS normalization forked from normalize.css;
  • A newly developed Markup Language;
  • A syntax highlighter from highlight.js
  • An icon font type from glyphicons provided as a part of bootstrap
  • An emoji font type from emoji symbols
  • Color themes of a white and black style.更换主题
